# Flowly - Workflow Automation Platform

A modern, responsive landing page for Flowly, an intelligent workflow automation platform.

## Project Structure

```
.
├── index.html              # Main HTML file
├── css/
│   ├── base/              # Base styles (reset, variables, typography)
│   ├── layouts/           # Layout components (container, grid, header)
│   ├── components/        # Reusable components (buttons, cards, badges)
│   ├── sections/          # Section-specific styles
│   ├── utilities/         # Utility classes (spacing, flex, visibility)
│   └── main.css           # Main stylesheet (imports all)
├── js/
│   └── main.js            # JavaScript functionality
├── assets/
│   ├── icons/             # SVG icons
│   ├── images/            # Images and product visuals
│   └── fonts/             # Custom fonts (optional)
└── README.md              # This file
```

## Getting Started

1. Open `index.html` in your browser
2. Add content and styles to the respective CSS files
3. Enhance interactivity with JavaScript in `js/main.js`

## Development Tips

- Use CSS variables in `base/variables.css` for colors, fonts, and spacing
- Keep component styles modular in `components/`
- Add section-specific styles in `sections/`
- Use utility classes for common patterns

## Features

- Responsive design
- Modular CSS architecture
- SVG icon system
- Mobile-first approach
